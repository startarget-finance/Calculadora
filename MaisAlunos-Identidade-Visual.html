09 · Uso Correto
Do & Don't
✓ Faça
Use UPPERCASE para títulos, section labels, botões e badges
Use border-radius máximo de 4px em todos os elementos
Use gap de 2px para criar grids "zero-gap" que parecem uma grade
Use laranja (#FF6500) APENAS como acento — nunca como background de seção completa (exceto CTA final)
Alterne fundos entre #000 e #0a0a0a para criar ritmo visual
Mantenha a hierarquia: section-label → section-title → body text
Use sempre font-weight: 900 para títulos e CTAs
Garanta espaço de 120px no topo da hero por causa do nav fixo
✗ Não faça
Não use a cor teal/verde (#00b4a6) — foi aposentada
Não arredonde elementos com border-radius acima de 4px (sem pills/100px)
Não use degradês (gradientes) de fundo — backgrounds são sempre sólidos
Não use box-shadow colorido em elementos (sem glow effects teal/verde)
Não coloque dois CTAs primários na mesma linha
Não use cores secundárias (azul, roxo) na UI — apenas para dados/gráficos
Não centralize o texto dos títulos principais (hero e sections são left-aligned)
Não use padding inferior a 60px nas laterais em desktop
