# GUIA DE DESIGN PARA O CANVA

## Paleta de Cores
| Cor | Uso | Hex sugerido |
|---|---|---|
| Preto | Fundo principal | #0A0A0A |
| Azul escuro | Fundo secundário | #0D1B2A |
| Verde-água | Destaques e títulos | #00B4A6 |
| Branco | Texto principal | #FFFFFF |
| Cinza claro | Texto secundário | #CCCCCC |

## Fontes sugeridas (disponíveis no Canva)
| Uso | Fonte |
|---|---|
| Títulos grandes | Montserrat Bold ou Bebas Neue |
| Subtítulos | Montserrat SemiBold |
| Texto corrido | Montserrat Regular |
| Destaques em badge | Montserrat ExtraBold |

## Tamanho dos slides
- **Apresentação padrão:** 1920 × 1080px (16:9)
- **Para enviar por WhatsApp/Instagram:** 1080 × 1080px (quadrado)

## Elementos visuais recomendados (Canva gratuito)
- Fotos de corrida ou jiu-jitsu (use o banco do Canva — filtre por "running", "jiu jitsu", "martial arts")
- Ícones lineares brancos para os bullets
- Formas geométricas simples (retângulos com bordas para badges)
- Overlay escuro sobre fotos: retângulo preto com 60-70% de opacidade
